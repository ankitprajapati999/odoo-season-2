# 🚀 Odoo Season 2

A modern React + Vite web application built with a clean UI, authentication, state management, and responsive design.

---

## 📖 Overview

Odoo Season 2 is a frontend application developed using modern React technologies. The project focuses on performance, scalability, reusable components, authentication, and responsive user experience.

---

## ✨ Features

- 🔐 User Authentication (Clerk)
- ⚡ Fast development with Vite
- 🎨 Responsive UI
- 📱 Mobile Friendly Design
- 📊 Interactive Charts (Recharts)
- 📝 Form Validation (React Hook Form + Zod)
- 🔄 State Management (Zustand)
- 🚀 React Router Navigation
- 🔔 Toast Notifications
- 🌙 Modern Component Architecture

---

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| React 19 | Frontend Framework |
| Vite | Build Tool |
| Tailwind CSS | Styling |
| Clerk | Authentication |
| React Router DOM | Routing |
| Zustand | State Management |
| Axios | API Requests |
| React Hook Form | Form Handling |
| Zod | Validation |
| Recharts | Charts |
| Lucide React | Icons |
| Sonner | Toast Notifications |

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/odoo-season-2.git
```

Go into the project

```bash
cd odoo-season-2
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

---

# 📦 Build for Production

```bash
npm run build
```

Preview Production Build

```bash
npm run preview
```

# 🌐 Environment Variables

Create a `.env` file in the project root.

Example:

```env

VITE_SUPABASE_URL=your_supabase_url

VITE_SUPABASE_ANON_KEY=your_supabase_key
```
